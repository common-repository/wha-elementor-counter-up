WHA Counter UP widget for Elementor allows you to add different animated widgets to the site. Easy to Install and Use.

== Description ==

WHA Counter UP is an easy to use jQuery based responsive plugin for Elementor which can be used in the website to showcase the number of clients, satisfied customers or anything related in a count down.

== Installation ==

1. Activate Plugin Elementor.
2. Activate Plugin WHA Elementor Counter Up.
3. Open the created page in Elementor mode.
4. Select Item Categories WHA WIDGETS.
5. Add widget to page.
